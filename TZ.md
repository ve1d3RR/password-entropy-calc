# Техническое задание: Интеллектуальная система аудита криптостойкости паролей 

## 1. Цель проекта
Разработка высокотехнологичного веб-сервиса для оценки защищенности учетных записей. Система объединяет математический анализ (энтропия Шеннона), моделирование аппаратных атак (пулинг GPU) и возможности искусственного интеллекта (GigaChat AI) для предоставления экспертных рекомендаций по безопасности.

## 2. Роли пользователей
* **Аудитор (Авторизованный):** Формирование сложных вычислительных кластеров, выбор целей атаки и получение расширенной AI-аналитики.
* **Администратор:** Управление справочниками оборудования и мониторинг логов.

## 3. Модели данных
* **TargetGroup (Группа цели):** Категоризация объектов (Финансы, Критическая инфраструктура и т.д.).
* **HardwareRig (Оборудование):** База данных GPU с хэшрейтами MD5/SHA-256 и показателями энергопотребления.
* **PasswordAuditLog (Журнал):** Хранение результатов аудита, включая состав кластера и оценки сложности.

## 4. Ключевой функционал
* **Constructor (Кластерный сборщик):** Интерфейс добавления множества разнородных видеокарт с индивидуальным указанием их количества. Суммирование мощностей в реальном времени.
* **Fail-safe GPU Search:** Гибридный поиск оборудования, использующий Minerstat API v2.0 (через X-API-Key) с автоматическим переключением на локальный кэш при сбоях сети.
* **AI Security Agent:** Интеграция с GigaChat API для генерации динамических фактов об ИБ и персональных советов по укреплению парольной политики.
* **Visual Analytics:** Интерактивные графики (линейный прогноз и круговая диаграмма вклада мощностей) с использованием Chart.js.

## 5. Внешние интеграции
* **GigaChat API (Sber):** Генерация интеллектуального контента и рекомендаций.
* **Minerstat API:** Получение актуальных спецификаций графических ускорителей.
* **Have I Been Pwned API:** Проверка паролей по базам утечек через протокол k-Anonymity.
