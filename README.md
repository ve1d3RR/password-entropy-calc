> **Password Entropy Calc** — это продвинутое веб-приложение на базе Django, предназначенное для проведения комплексного аудита безопасности паролей. Сервис сочетает в себе математические расчеты энтропии Шеннона, моделирование аппаратных атак на кастомных GPU-кластерах и интеллектуальную аналитику от нейросети GigaChat.

## Ключевые функции

*   **Комплексная система аудита паролей:** Управление логами аудита (`PasswordAuditLog`) для отслеживания и оценки безопасности учетных записей. Система позволяет фиксировать детали проверок и выявлять потенциальные уязвимости.
*   **Конструктор вычислительного кластера:** Уникальный интерфейс, позволяющий объединять различные видеокарты в единый пул. Пользователь может указать индивидуальное количество для каждой модели оборудования, а система автоматически суммирует их мощность для расчета времени взлома.
*   **Интеллектуальный поиск GPU:** Интеграция с **Minerstat API v2.0**. Позволяет находить спецификации видеокарт в глобальной базе и автоматически заполнять данные о хэшрейте (MD5/SHA-256). Реализована система локального кэширования для защиты от лимитов API.
*   **AI Security Consultant (GigaChat):** Интеграция с ИИ от Сбера. Система генерирует динамические факты об информационной безопасности и предоставляет персональные экспертные советы по улучшению каждого проверенного пароля.
*   **Безопасный аудит утечек (k-Anonymity):** Проверка пароля по базе *Have I Been Pwned* через защищенный протокол. Во внешнюю сеть передается только часть SHA-1 хэша, что гарантирует полную конфиденциальность данных.
*   **Расширенная визуализация:** Использование **Chart.js** для построения динамических графиков устойчивости пароля и круговых диаграмм распределения мощностей внутри кластера.

##  Обзор архитектуры

Проект реализован как монолитное веб-приложение с использованием паттерна **MVT (Model-View-Template)**:

*   **Model:** Определены в `audit/models.py`. Сущности `PasswordAuditLog`, `HardwareRig` и `TargetGroup` формируют основу системы аудита и связей Many-to-Many.
*   **View:** Находятся в `audit/views.py`. Отвечают за обработку HTTP-запросов, взаимодействие с ИИ и API поиском видеокарт.
*   **Template:** Находятся в `audit/templates`. Обеспечивают структуру HTML и презентационную логику.
*   **Django Admin:** Мощный интерфейс управления в `audit/admin.py` для быстрого редактирования справочников оборудования и групп целей.
*   **Services:** Бизнес-логика (расчет энтропии, форматирование времени, OAuth авторизация GigaChat) вынесена в `audit/services.py`.

##  Инструкция по запуску

1.  **Клонирование репозитория:**
    ```bash
    git clone https://github.com/ve1d3RR/password-entropy-calc.git
    cd password-entropy-calc
    ```

2.  **Настройка виртуального окружения:**
    ```bash
    python -m venv venv
    # Windows:
    .\venv\Scripts\activate
    # macOS/Linux:
    source venv/bin/activate
    ```

3.  **Установка зависимостей:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Установка SSL-сертификатов (для GigaChat):**
    ```bash
    python -c "import certifi, requests; r = requests.get('https://gu-st.ru/content/lending/russian_trusted_root_ca_pem.crt', verify=False); open(certifi.where(), 'a', encoding='utf-8').write('\n' + r.text)"
    ```

5.  **Миграции и создание администратора:**
    ```bash
    python manage.py migrate
    python manage.py createsuperuser
    ```

6.  **Запуск:**
    ```bash
    python manage.py collectstatic
    python manage.py runserver
    ```

##  Структура файлов

*   `audit/`: Ядро приложения. Содержит модели данных, логику ИИ и шаблоны конструктора кластера.
*   `config/`: Глобальные настройки проекта Django (settings, urls).
*   `static/`: Кастомные ассеты (логотипы, иконки).
*   `staticfiles/`: Собраные статические файлы для работы сервера.
*   `manage.py`: Утилита командной строки Django.
*   `requirements.txt`: Список зависимостей проекта.
